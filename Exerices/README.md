# 西电电院数据结构上机

该项目为电院上机题目及解析。统一规范，所有文件夹名、文件名都将采用`序号+英文`格式。

同时为加强可读性，每种结构的定义、基本操作都放在本目录下同一个头文件中。每道题用一个C文件完成。

至于解析，取决于本人精力和空闲时间了。**当然如果你有想法，欢迎和我一起完成它！**

上机分为3次，内容及说明分别如下：
